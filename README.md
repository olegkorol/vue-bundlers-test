# Quick Bundlers Test

> A very quick and dead simple comparison between Parcel, Rollup and Webpack.

## Builds

``` bash
# install dependencies
yarn

# build with webpack (to dir: wpck/)
yarn wbpck

# build with rollup (to dir: rllp/)
yarn rllp

# build with parcel (to dir: dist/)
yarn prcl
```
